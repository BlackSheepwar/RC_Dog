# RC Dog

四足机器人控制程序集 — 基于 STM32 平台的遥控四足机器人项目。

## 项目概览

本项目包含四个独立的 MCU 程序，分别承担不同的角色：

| 程序 | 平台 | 角色 | 状态 |
|------|------|------|------|
| [DogCore](DogCore/) | STM32H750VBT6 | **四足狗主控核心** — 控制机械狗本体运动 | 活跃开发 |
| [RemoteControl](RemoteControl/) | STM32F103C8T6 | **遥控器** — 用于控制四足狗 | 活跃开发 |
| [dogF103C8T6](dogF103C8T6/) | STM32F103C8T6 | **舵机角度控制器** — 调整舵机角度、归零校准 | 维护中 |
| [FrontDog](FrontDog/) | STM32F405RGT6 | **第一代舵机狗控制器**（架构验证版） | 已存档，不再更新 |

---

### DogCore — 四足狗主控核心

- **芯片**: STM32H750VBT6
- **职责**: 四足机器人的主控制核心，负责运动控制、姿态解算、通信管理等核心功能
- **定位**: 当前的主力开发项目，承载机械狗本体控制逻辑

### RemoteControl — 遥控器

- **芯片**: STM32F103C8T6
- **职责**: 遥控器端程序，负责采集遥控指令并通过无线通信发送给主控
- **定位**: 与 DogCore 配合使用的控制终端

### dogF103C8T6 — 舵机角度控制器

- **芯片**: STM32F103C8T6
- **职责**: 专门的舵机调试工具，用于调整舵机角度、执行归零操作、设定校准值
- **用途**: 在装配调试阶段对每个舵机进行独立角度校准

### FrontDog — 第一代舵机狗控制器（已存档）

- **芯片**: STM32F405RGT6
- **职责**: 最早基于 F405 平台制作的第一代舵机狗控制器，完成了架构验证
- **状态**: **已存档，不再更新。** 其功能与经验已由 DogCore 继承和迭代

---

## 历史沿革

```
FrontDog (F405)  ── 第一代验证 →  经验继承
                                     ↓
dogF103C8T6      ── 舵机调试工具     DogCore (H750) ── 当前主控核心
                                     ↑
RemoteControl    ── 遥控器端 ────────┘
```
