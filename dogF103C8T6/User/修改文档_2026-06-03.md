# 修改文档 — 2026-06-03

## 一、新增功能：多舵机光标控制 + 电位器锁定

### 1.1 概述
为 3 个舵机引入**光标选择**机制，通过按键在舵机间切换控制目标，并支持**锁定电位器**以实现微调。

### 1.2 按键功能表

| 按键 | 操作 | 功能 |
|------|------|------|
| **键1** | 短按 | 当前光标舵机角度 **-1°** |
| **键2** | 短按 | 光标移到**下一舵机**（1→2→3→1…） |
| **键3** | 短按 | 电位器 **开关切换**（默认关闭，第一次按下打开） |
| **键4** | 短按 | 光标移到**上一舵机**（1→3→2→1…） |
| **键4** | 长按 | **锁定/解锁**电位器控制 |
| **键5** | 短按 | 当前光标舵机角度 **+1°** |

### 1.3 OLED 显示格式

使用 `afont16x8` 字体（16px × 8px），每行 16px 高：

```
>1:+045     <- 第1行(y=0)  光标所在舵机，前缀 >
 2:-023     <- 第2行(y=16) 非光标舵机，前缀空格
 3:+112     <- 第3行(y=32) 非光标舵机
K:OFF       <- 第4行(y=48) 电位器状态 + 锁定指示
```

**第4行状态指示：**

| 显示 | 含义 |
|------|------|
| `K:OFF` | 电位器**关闭**（默认状态） |
| `K:ON`  | 电位器**开启**，未锁定 |
| `LOCK`  | 电位器**开启**但已**锁定**（旋钮无效） |

角度格式：`%+04d` — 固定 4 位，带符号，前导零填充（如 `+045`、`-023`）。

显示优化：使用 `OLED_ShowFrameArea(0, 63, 0, 63)` 局部刷新，只更新有效内容区域（列0~63，行0~63），减少 SPI 传输量。

## 三、新增模块 API 文档

### 模块路径：`User/APP/APP_Cursor/`

#### 结构体 `APP_Cursor_t`

```c
typedef struct {
    uint8_t index;          // 光标索引（0~2）
    uint8_t current_id;     // 当前选中的舵机 ID（1/2/3）
    uint8_t lock;           // 电位器锁定：1=锁定，0=启用
} APP_Cursor_t;
```

#### 全局变量 `g_cursor`

所有模块均可通过 `extern APP_Cursor_t g_cursor;` 直接读取当前光标状态。

#### 函数接口

```c
void APP_Cursor_Init(void);
    // 初始化：index=0, current_id=1, lock=0

void APP_Cursor_Next(void);
    // 光标前移：1→2→3→1…

void APP_Cursor_Prev(void);
    // 光标后移：1→3→2→1…

void APP_Cursor_ToggleLock(void);
    // 反转锁定状态
```

## 四、关键逻辑说明

### 4.1 电位器控制流程（`tsak_se.c`）

```
for(;;)
{
    if (!g_cursor.lock)                     // 未锁定
    {
        if (APP_KNOB_GetValue(1, &val) == 0)  // 电位器需已打开
        {
            APP_SE_SetTarget(g_cursor.current_id, val);  // 控制光标舵机
        }
    }
    APP_SE_Scheduler();
    osDelay(10);
}
```

- **电位器关闭**（`K:OFF`）：`GetValue` 返回 `-3`，旋钮无效
- **电位器打开 + 未锁定**（`K:ON`）：旋钮控制光标舵机
- **电位器打开 + 锁定**（`LOCK`）：旋钮数据被丢弃，舵机保持当前位置

### 4.2 锁定作用域

锁定是**全局**的：锁定后所有舵机均不受旋钮控制。切换光标不会自动解锁，需再次长按键4解除锁定。

### 4.3 电位器开关初始化

- 注册时 `enable = 0`（默认关闭）
- 键3 第一次按下：`a=1` → `SetEnable(1, 1)` → **打开**
- 键3 第二次按下：`a=0` → `SetEnable(1, 0)` → **关闭**
- OLED 第4行实时显示 `K:OFF` / `K:ON` / `LOCK`

## 五、验证方法

| 测试 | 预期结果 |
|------|----------|
| 上电 | OLED 显示 `>1:+000 2:+000 3:+000`，第4行 `K:OFF` |
| 旋钮转动 | **无效**（电位器默认关闭） |
| 按键3 短按 | 第4行变为 `K:ON`，旋钮开始控制舵机1 |
| 旋钮转动 | 舵机1 跟随旋钮动作，舵机2/3 不动 |
| 按键2 短按 | 光标移到舵机2（`>2:+000`），旋钮控制舵机2 |
| 按键4 短按 | 光标移回舵机1 |
| 按键1 短按 | 光标舵机角度 -1° |
| 按键5 短按 | 光标舵机角度 +1° |
| 按键4 长按 | OLED 第4行变为 ` LOCK`，旋钮不再控制任何舵机 |
| 按键4 再次长按 | ` LOCK` 消失，恢复 `K:ON`，旋钮恢复控制 |
| 按键3 再次短按 | 第4行变为 `K:OFF`，旋钮无效 |
